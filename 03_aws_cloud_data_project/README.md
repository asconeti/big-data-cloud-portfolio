# AWS Cloud Data Project

This folder contains a cloud-oriented Big Data assignment focused on implementing a small **data architecture on AWS**.

## Main topics covered

- Cloud foundations and AWS services
- Data storage and processing workflows
- Event-driven logic with Lambda
- Compute resources with EC2
- Relational storage with RDS
- Monitoring with CloudWatch
- Architecture documentation in notebook format

## Main files

- `activitat_cloud_aws_CAT_Jordi_Maria.ipynb`: main notebook with the cloud project
- `activitat_cloud_aws_CAT_Jordi_Maria.html`: exported HTML version of the notebook
