# Big Data & Cloud Portfolio

This repository showcases a selection of **Big Data and cloud computing projects** developed as part of my academic training in the **Data Science Master's program at UOC**.

The portfolio is designed to highlight practical work related to **distributed data processing**, **streaming analytics**, and **cloud-based data architectures**, combining analytical thinking with hands-on experience in tools and concepts commonly used in modern data platforms.

## Repository purpose

This repository is intended as a compact portfolio for recruiters, hiring managers, and technical professionals interested in my background in:

- Batch data processing
- Distributed file systems and large-scale analytics
- Streaming ingestion and real-time processing
- Apache Kafka and Spark Streaming
- Cloud-based data architectures on AWS
- Data pipelines and monitoring concepts
- Applied analytics in production-oriented environments

## Projects included

I recommend organizing the repository with these folder names:

### 1. `01_hdfs_batch_processing`
Batch-oriented project focused on **HDFS**, **PySpark RDDs**, **Spark SQL**, and the analysis of heterogeneous datasets, including CSV and JSON data sources.

### 2. `02_streaming_kafka_spark`
Streaming-oriented project focused on **Apache Kafka** and **Spark Streaming**, covering ingestion, producer-consumer workflows, and real-time data processing.

### 3. `03_aws_cloud_data_project`
Cloud-oriented project focused on the design and implementation of a small **AWS-based data architecture**, using services such as **S3**, **Lambda**, **EC2**, **RDS**, and **CloudWatch**.
