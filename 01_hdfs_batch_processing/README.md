# HDFS Batch Processing

This folder contains a Big Data assignment focused on **batch processing** and **distributed storage** concepts.

## Main topics covered

- HDFS file operations
- PySpark RDD transformations and actions
- Spark SQL queries
- Analysis of heterogeneous datasets
- Processing CSV and JSON sources
- Scalable data analysis workflows

## Main files

- `Actividad_Batch_Jordi_Baiges_Maria_Serra.ipynb`: main notebook with the practical work
- `consumo_hogar_2024.csv`: structured dataset used in the exercises
- `consumo_hogar_2024_validado.csv`: validated version of the dataset
- `tweets28a_sample.json/`: partitioned JSON dataset used for analysis
- `informe_Jordi.txt` and `informe_Maria.txt`: supporting notes and written outputs
