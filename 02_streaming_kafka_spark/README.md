# Streaming with Kafka and Spark

This folder contains a Big Data assignment focused on **streaming data ingestion** and **real-time analytics**.

## Main topics covered

- Apache Kafka fundamentals
- Producer-consumer workflows
- Event ingestion in streaming scenarios
- Spark Streaming integration
- Real-time data processing logic
- Python-based execution scripts for practical exercises

## Main files

- `Actividad_Streaming_CAT.ipynb`: main notebook with the activity
- `exercici_10_1.py` to `exercici_10_5.py`: Python scripts for Kafka-related exercises
- `exercici_11_1.py` to `exercici_11_4.py`: Python scripts for streaming and processing tasks

## Suggested folder rename

Recommended folder name for GitHub:

`02_streaming_kafka_spark`

## Notes

This project is especially useful to showcase familiarity with streaming concepts, distributed event processing, and notebook-plus-script workflows.
